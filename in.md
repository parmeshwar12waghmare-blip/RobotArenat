# RobotArenat

## Project structure
- src/ - main source code for the React app
- src/components/ - reusable UI components
- src/pages/ - page-level components
- src/data/ - static data used by the app
- public/ - static files
- package.json - project scripts and dependencies

## How to run
1. Install Node.js and npm (see requirements.txt).
2. Extract the project folder.
3. Double-click start.bat to install dependencies, start the app, and open it in the browser.
4. If the browser does not open, open http://127.0.0.1:5173 manually.
